1. Introduction and Motivation

Transceiver–>access point(AP)

user device–>user equipment(UE)

cellular networks : a UE connects to one AP

cellular networks的问题：large variations between center and edge

Goal : 不只是提高峰值速率，而是提高覆盖范围内的所有速率

Network-Centric Clustering(网络中心集群)

  1. each UE is affected by interference from own cell and neighbore cells

  2. UEs close to the cluster edge not benefit from the network-centric clustering(簇边缘的用户不能获得增益)

  3. DCC : dynamic cooperation clustering(动态合作集群)

2. User-Centric Cell-Free Massive MIMO Networks

2.1 Definition of Cell-Free Massive MIMO

ultra-dense network( many more APs than UEs) where the APs are cooperating to serve the UEs by joint coherent transmission and reception

L APs with N antennas

total number of AP antennas M = NL

K single-antenna UEs : L>>K and M>>K

!! no need for a formal assumption of M>>K or L >>K !!

2.2 User-Centric Dynamic Cooperation Clustering

if AP reach the UE with a signal power is non-negligible compared to the thermal noise power take an active part in serving that UE, then the performance loss should be negligible.

Definition 2. 1 : Dynamic cooperation clustering (DCC)

UE k is served only by the APs with indices in the set $M_k$

Assume the sets $M_k$ is fixed : dynamic variations occur over much larger time intervals than channel fading variations(假设Mk是固定的：原因是动态变化发生的时间间隔比信道衰落变化大得多。)

diagonal matrices (对角矩阵): $D_{kl}\in \bold C^{N\times N}$

2.3 System Models for Uplink and Downlink

2.3.1 Block Fading and Coherence Blocks

Wireless channels are linear(due to the superposition principle(叠加原理) by Maxwell’s equation) but time-varing（movable）

Channel Coherence Time

the channel can be approximated as constant–>the communication system is time-invariant

  1. within the coherence time, channel can be described by a finite impulse response(FIR) filter : each term of the impulse response describes one distinguishable propagation path in a multipath environment with a distinct time delay and pathloss.

  2. reacts different to different frequencies of input signals but the variations are smooth

  3. channel coherence bandwidth: a sufficiently narrow frequency range the channel can be considered constant

Channel coherence block

a time-frequency block :

|– time dration = coherence time

|– frequency width = coherence bandwidth

Block-fading model

$T_c \longrightarrow coherence\quad time$

$B_c \longrightarrow coherence \quad bandwidth$

$\tau_c = T_cB_c \longrightarrow transmission \quad symbols$

Factors of Tc and Bc :

  1. channel delay spread : the time difference between the shortest and longest resolvable path
  2. UE mobility
  3. carrier frequency

Approximation of Coherence time :$T_c = \lambda/4v$

Approximated of Coherence bandwidth : $B_c = 1/2T_d$

Td is channel delay spread –> the time difference betwwen the earliest and last propagation path

2.3.2 Time-Division Duplex Protocol(时分双工协议)

  1. Each coherence block is used for both uplink and downlink transmissions.
  2. Tranmit pilots only in the uplink–>channel reciprocity
  3. Using pilots Ap can estimate the channels

$$
\tau_c = \tau_p+\tau_u+\tau_d
$$

–Transmit uplink data before the iplink pilots : increases the latency–> data detection cannot be initiated until after the pilots been received(前一种变体的缺点是等待时间增加，因为直到接收到导频之后才能启动数据检测。)

–change the order between the uplink and downlink data : requires a guard time interval – > make sure all UEs receive the downlink data before they switch to uplink transmission mode (变化是下行链路和上行链路之间的切换需要 保护时间间隔，使所有UE先接收到下行数据 他们切换到上行链路传输模式。)

2.4 Network Scalability(网络可扩展性)

Conventional cellular networks achieve scalability through a divide-and-conquer approach.（传统蜂窝网络使用分而治之的方法实现可扩展性）

cellular networks are inherently scalable(蜂窝网络生来可扩展)

Definition 2.2 (Scalability)

A Cell-free Massive MIMO network is scalable if all the four above-listed tasks have finite computational complexity and resource requirements with respect to each AP as K->$\infin$

2.4.1 Example of Unscalable Network Operation

  1. complexity of signal processing for channel estimation
  2. AP need to create the downlink signal $\sum_{i=1}^{K}w_{il}\varsigma_i$
  3. AP need to forword its K processed received signals over the fronthaul links
  4. selet how to allocate its transmit power betwwen the K UEs.

NLoS 随机信道模型： uncorrelated Rayleigh fading（非相关瑞利衰落信道）

  2. 接收信号是大量具有随机相位移动的传播路径的总和

  3. 方差$\beta_{kl}$表示大尺度衰落和平均信道质量当UE在阿哥相对小的范围移动–>由路径损耗和阴影衰落决定

2.6 Channel Hardening and Favorable Propagation(信道硬化和最优传播)

Channel Hardening and Favorable Propagation are not depends strongly on the number of antennas and the spatial channel correlation properties.

2.6.1 Channel Hardening

effective channel

  1. 有效信道的值几乎保持不变，即使L个信道向量是改变的
  2. 当有效信道的随机实现接近平均值（例如方差很小），那么相干块中有相同有效标量信道，于是我们可以使用确定性信道来操作他。（信道硬化）

Definition 2.3 Channel Hardening

for a given set $M_k$ of serving APs and downlink power allocation coeffcients$\rho_{k1},…,\rho_{kL}$ the effective channel is asymptotic channel hardening if : (when N$\rightarrow \infin$ )

**Lemma 2.3 ** for correlated Rayleith fading channels, a sufficient condition for the effective channel to provide asymptotic channel hardening is

  1. $tr(R_{kl}^2)$ can be computed as the sum of the sqquared eigenvalues of the spatial correlation matrix $R_{kl}$
  2. $tr(R_{kl}) = N \beta_{kl}$
  3. $tr(R_{kl}^2 ) = N\beta_{kl}^2$ when all eigenvalues equal to $\beta_{kl}$
  4. tr($R_{kl}^2) = N^2\beta_{kl}^2$ when only one non-zero eigenvalue that equals N$\beta_{kl}$
  5. the value need to be small and eigenvalues are the same achieves the smallest value

Impact of the Numer and Geographical Distribution of APs

假设$R_{kl} = \beta_{kl}I_N$，即空间相关矩阵中没有空间相关性，那么上述式子就会变为：

  1. 如果所有服务的AP有相同的$\rho_{kl}\beta_{kl}$的值，那么上式会变为$1/(N|M_k|)$ –>可以看到，在信道硬化上拥有很多AP和AP有很多天线会有同样的效果

  2. 如果所有服务的AP有不同的值，那么相同程度信道硬化所需的天线数也会高于蜂窝大规模MIMO

  3. 距离决定大尺度衰落系数{$\beta_{kl}$}为了保证服务的AP有近似的$\rho_{kl}\beta_{kl}$，那么我们可以减少距离较近AP的功率分配(对$\beta_{kl}$较大的减少$\rho_{kl}$)

  4. 第三点提到的方法虽然可以增加信道硬化程度，但不是很理想的，我们需要获得大的接收端UE 的接收功率通过为距离近的AP分配更高的功率

Takeaways

  1. 对于无蜂窝网络而言信道硬化是理想属性，不是强制要求

  2. 无蜂窝网络的信道硬化程度比蜂窝网络低

  3. 信道硬化使得某些容量接近真实容量
